Centos7安装docker,并添加国内源 ​

前言 ​

学校工作室的机子总是时不时会坏一下，要重装环境，这里记录一个docker的安装防止忘记。

环境要求：

  • 一台安装了 CentOS 服务器或虚拟机。
  • 系统需要能够访问互联网。
  • 推荐使用 root 账户或拥有 sudo 权限的用户来执行安装过程。

步骤 ​

1、如果系统已安装docker,删除原有docker ​

bash
yum remove docker \
    docker-client \
    docker-client-latest \
    docker-common \
    docker-latest \
    docker-latest-logrotate \
    docker-logrotate \
    docker-selinux \
    docker-engine-selinux \
    docker-engine \
    docker-ce

2、更新yum包 ​

bash
sudo yum update -y

3、安装必要的包 ​

bash
sudo yum install -y yum-utils device-mapper-persistent-data lvm2

4、设置yum镜像源，下载docker ​

bash
sudo yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
sudo yum install docker-ce docker-ce-cli containerd.io

5、设置docker国内源 ​

使用vi或vim打开输入内容

bash
sudo vim /etc/docker/daemon.json

输入docker源内容保存

最近国内一些docker源关闭了，这里只示例一个试了可以用的，其他源可以参考docker镜像加速源配置

json
{
    "registry-mirrors": [
        "https://docker.registry.cyou"
    ]
}

重新加载daemon并重启 Docker 服务

bash
sudo systemctl daemon-reload
sudo systemctl restart docker

输入以下命令，如果打印的是刚才json里配置的内容则证明配置成功

bash
docker info

6、验证是否安装成功 ​

输入以下命令，运行helloworld样例

bash
sudo docker run hello-world

安装成功则会打印出 Hello world from Docker

7、设置自启动 ​

bash
sudo systemctl enable docker

至此docker的安装已全部完毕
